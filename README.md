# 🐮 Search For Cattles

一个基于抖音**佛系消消消**玩法的，在浏览器中创建和求解 "Cattles" 谜题的交互式网页应用。

## 📖 使用方法

1. 用浏览器打开 `index.html`
2. 在左上角输入框中设置棋盘边长 n（2~15），点击「生成棋盘」
3. 点击下方调色板选择颜色，在棋盘上**左键拖拽**涂色
4. 滚动**鼠标滚轮**可快速切换颜色；点击「🧹 擦除」可擦除已涂格子
5. 涂满所有格子后，点击「验证拼图」检查是否合法（颜色数 = n、每种颜色连通）
6. 验证通过后点击「求解」，等待 🐮 解答出现
7. 点击「清空」可重置棋盘重新开始

## 🎮 玩法 (与抖音相同)

1. 📐 输入棋盘边长 n（2~15），生成 n×n 空白棋盘
2. 🎨 选择颜色，在棋盘上拖拽涂色，划分 n 个彩色区域
3. 📏 规则：
   - 每种颜色的格子必须四方向连通
   - 所有格子必须涂满
   - 颜色种类恰好为 n
   - 同色被分割时，小块自动清除；被完全包围的区域自动填充
4. 🐮 验证通过后点击求解，棋盘上显示每色一头牛的摆放方案（不同行、不同列、不斜对角）

## 🖱️ 操作

| 操作 | 方式 |
|------|------|
| 🎨 涂色 | 左键点击/拖拽 |
| 🧹 擦除 | 点击"擦除"按钮后拖拽 |
| 🔄 切换颜色 | 点击调色板 / 滚轮滚动 |
| ✅ 验证拼图 | 点击"验证拼图" |
| 🔍 求解 | 验证通过后点击"求解" |
| 🗑️ 清空 | 点击"清空" |

## 🛠️ 技术

纯前端，零依赖。打开 `index.html` 即可使用。

## 📁 项目结构

```
├── index.html          ← 入口页面
├── css/
│   └── style.css       ← 暗色主题样式
└── js/
    ├── state.js        ← 状态管理
    ├── renderer.js     ← Canvas 棋盘渲染
    ├── swatches.js     ← 调色板
    ├── auto-process.js ← 连通性检查 & 包围填充
    ├── interaction.js  ← 鼠标/触屏交互
    ├── validator.js    ← 拼图验证
    ├── solver.js       ← 求解器 (Web Worker)
    └── main.js         ← 初始化 & 按钮绑定
```
